# Site Background Image

Place your background image file here and name it `site_bg.png`.

To use a different background image:
1. Place your image file in this directory (e.g., `zen_bg.jpg`, `sand_rocks.jpg`, etc.)
2. Update the CSS variable in `src/App.vue`:
   - Find: `--site-background-image: url('/images/site_bg.png');`
   - Change to: `--site-background-image: url('/images/your-image-name.jpg');`

Supported formats: PNG, JPG, JPEG, WEBP, GIF

Recommended size: 1920x1080 or larger for best coverage
